# k8s-ops-toolkit
Suite d'outils CLI en Bash pour l'automatisation de la supervision et de l'administration Kubernetes multi-cluster.

> 🚧 Projet en cours de développement.

## Outils prévus
- `cluster-health.sh` — diagnostic de santé des nodes et pods
- `wait-for-rollout.sh` — attente d'un déploiement
- `log-cleaner.sh` — nettoyage standardisé des logs
- `deploy-notify.sh` — notifications Slack/Discord
